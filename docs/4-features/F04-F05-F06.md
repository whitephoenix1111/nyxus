# F04 — Clients Page

> **Route**: `/clients`
> **Accessible by**: Salesperson (`ownerId === me` & `isProspect = false`) · Manager (tất cả)

---

## Mục tiêu

Danh sách khách hàng đã qualify (isProspect = false) — xem, sửa thông tin, xem toàn bộ lịch sử deal và tài liệu.

---

## Hiển thị & Lọc

- Grid view — mỗi card là 1 Client với stats tổng hợp
- Chỉ hiện `isProspect = false` và không có `archivedAt`
- `<OwnerFilter />` — chỉ hiện với Manager
- Search theo tên / công ty
- Filter theo industry / tag

---

## ClientCard

| Field | Ghi chú |
|---|---|
| Avatar, Name, Company | — |
| Total Value | SUM tất cả opportunities |
| Opportunity Count | — |
| Top Status | Status của deal lớn nhất |
| Tags | Computed + DB tags |
| `<OwnerBadge />` | Manager only — góc dưới phải |

---

## Detail Panel (slide-over từ phải)

Click card → mở `<DetailPanel />`:

**Tab "Cơ hội"**
- Danh sách opportunities của client
- Mỗi opp: value, status badge, confidence, lastContactDate
- `canEdit` guard: Edit/Delete ẩn nếu salesperson non-owner

**Tab "Tài liệu"**
- Documents của client (filter theo clientId)
- `canEdit` guard tương tự

---

## Thêm mới

**Import khách hàng cũ** — `<ExistingClientModal />`:
```
POST /api/clients/existing
  → Client (isProspect: false) + Opportunity (Won, confidence: 100%)
  → Dùng cho onboarding base khách hàng đã có
```

---

## Soft Delete (Manager only)

```
DELETE /api/clients/[id]
  → Confirm dialog nếu có deal đang mở
  → archivedAt = today
  → Cascade xóa: opps chưa Won + tasks pending
  → Giữ: activities, tasks done, opps Won
```

---

## Acceptance Criteria

- [x] Grid chỉ hiện isProspect=false, không có archivedAt
- [x] ClientCard có OwnerBadge (manager only)
- [x] DetailPanel 2 tab: Cơ hội + Tài liệu
- [x] canEdit guard ẩn Edit/Delete với non-owner salesperson
- [x] Import khách hàng cũ tạo đúng Client + Opportunity(Won)
- [x] Soft delete chỉ Manager, có confirm dialog
- [x] Cascade đúng: giữ Won opps và done tasks

# F05 — Forecast Page

> **Route**: `/forecast`
> **Accessible by**: Manager only — salesperson redirect về `/`

---

## Mục tiêu

Dự báo doanh thu dựa trên weighted confidence. Không thay đổi data — chỉ đọc và visualize.

---

## Tính toán

```
Weighted Revenue = SUM(opportunity.value × opportunity.confidence / 100)
```

Áp dụng cho tất cả opportunities trừ Lost.

---

## UI Components

| Component | Mô tả |
|---|---|
| Header metric | Tổng Weighted Revenue |
| Funnel chart | Count + value mỗi stage |
| Table | Tất cả opps trừ Lost |
| Confidence column | Hiển thị %, fine-tune inline trong range của stage |

**Table columns**: Client · Company · Value · Confidence (%) · Weighted Value · Stage

---

## Acceptance Criteria

- [x] Redirect về `/` nếu role = salesperson
- [x] Weighted revenue = SUM(value × confidence/100) đúng
- [x] Funnel chart theo stage
- [x] Confidence fine-tune trong range — Qualified: 20–50%, Proposal: 45–75%, Negotiation: 70–90%
- [x] Lead (15%), Won (100%), Lost (0%) không fine-tune được

# F06 — Documents Page

> **Route**: `/documents`
> **Accessible by**: Salesperson (client.ownerId === me) · Manager (tất cả)

---

## Mục tiêu

Quản lý tài liệu liên quan đến khách hàng và deals. Metadata-only — không lưu binary.

---

## Hiển thị & Lọc

- Card / list view
- `<OwnerFilter />` — Manager only
- Filter: category · file type · search theo tên
- Star toggle để đánh dấu quan trọng

---

## Upload Document — `<UploadDocModal />`

```
1. Chọn client (searchable — chỉ thấy client của mình)
2. Chọn deal (optional, filter theo clientId)
3. Nhập: name, type (pdf/doc/xls/img/other), category (Hợp đồng/Đề xuất/Báo cáo/Hoá đơn)
4. url: null (placeholder) hoặc link ngoài

POST /api/documents → metadata vào documents.json
```

---

## Guard Rules

- **View**: Salesperson thấy docs có `client.ownerId === me`; Manager thấy tất cả
- **Mutate (PATCH/DELETE)**: Kiểm tra `client.ownerId`, không phải `document.ownerId`

---

## Acceptance Criteria

- [x] Salesperson chỉ thấy docs của client mình phụ trách
- [x] Manager thấy tất cả + OwnerFilter
- [x] UploadDocModal: client select searchable, chỉ thấy client của mình
- [x] Deal select filter theo clientId đã chọn
- [x] Star toggle PATCH isStarred
- [x] Delete guard: client.ownerId === me hoặc manager

# F03 — Opportunities Page

> **Status**: Phase 2  
> **Route**: `/opportunities`

---

## Mục tiêu

Overview toàn bộ pipeline — tất cả status, có thể filter và chuyển trạng thái.

## Acceptance Criteria

- [ ] Table view mặc định (có thể thêm Kanban view sau)
- [ ] Filter tabs: All | Lead | Proposal | Forecast | Order
- [ ] Sort by: Value (desc default), Date, Status
- [ ] Mỗi row có status badge màu khác nhau:
  - Lead: xám
  - Proposal: xanh dương mờ
  - Forecast: vàng mờ
  - Order: lime
- [ ] Click vào row → mở detail slide-over panel (từ phải)
- [ ] Detail panel: xem + edit tất cả fields + history log
- [ ] Bulk action: select multiple → update status
